Contoh Penggunaan Font Style dengan CSS

Paragraf pertama ini menggunakan ukuran font dalam satuan pixel, font-weight normal, dan jenis font Arial.

Paragraf kedua menggunakan ukuran font em, font-weight bold, serta gaya font italic dengan jenis font Times New Roman.

Paragraf ketiga memakai satuan rem untuk ukuran font, font-weight lighter, dan font-family Georgia.

Paragraf keempat menggunakan ukuran font persen, font-weight bolder, dan font-family Verdana.

Paragraf kelima menggunakan ukuran font pt, font-weight 600, serta font-family Courier New.
